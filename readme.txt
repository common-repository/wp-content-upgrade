== Description ==
Getting more newsletter signups can be a pain. With this plugin you can get many more. Write content as you normally do and offer a content upgrade. Insert code from Mailchimp or other sites inside your popup and see the increase in signups.

How to use:
After installing af activating the plugin, simply insert this shortcode:
You will find a custom post type called Content Upgrade, create your content upgrade popups.
When saving you will get more setting and your shortcode.
